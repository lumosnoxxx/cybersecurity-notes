# Installing Android Studio on my Kali Machine
(OK i actually haven't documented it yet but will do shortly !)
IT professionals might want to use Android Studio for various reasons. 
Could be developping Android Apps (quite obvious),
but for us Cybersecurity workers, it's one of the most valuable tools to audit and analyze behaviors, functionnalities and flaws of a mobile application. 

So, to get it : 
## Step 1 : Download Android Studio (always from official sources please) 
Get the official downloader on official android site [here](https://developer.android.com/studio)  
Get command line tools only at the end of the page, extract then put here : 
~/Android/Sdk/cmdline-tools/latest/bin/sdkmanager
~/Android/Sdk/cmdline-tools/latest/bin/avdmanager
